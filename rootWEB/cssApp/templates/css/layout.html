플롯팅 속성

float 속성은 특정요소를 떠 있게 만들어 줍니다. 여기서 '떠 있다 '라는 말의 의미는 특정 요소가 기본 레이아웃에서 벗어나 문서의 왼쪽 또는 오른쪽에 이동하는 것입니다. float 속성은 복잡한 형태의 레이아웃을 구성하는데 피룡한 핵심 속성이며, 특정 요소가 주변 요소와 자연스럽게 어울리게 해 줍니다.
